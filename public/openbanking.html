4. API Usage Examples

Once consent is granted, you can access the user's financial data:

Get Accounts

curl -X GET http://localhost:3000/openbanking/v1/accounts \ -H "Authorization: Bearer YOUR_TOKEN" \ -H "x-consent-id: YOUR_CONSENT_ID"

Get Transactions

curl -X GET http://localhost:3000/openbanking/v1/accounts/acc_001_001/transactions \ -H "Authorization: Bearer YOUR_TOKEN" \ -H "x-consent-id: YOUR_CONSENT_ID"

Check Funds

curl -X POST http://localhost:3000/openbanking/v1/funds-confirmation \ -H "Authorization: Bearer YOUR_TOKEN" \ -H "x-consent-id: YOUR_CONSENT_ID" \ -d '{"accountId":"acc_001_001","amount":1000}'
